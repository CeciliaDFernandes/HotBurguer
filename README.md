🤝Sobre o Projeto
O HotBurguer é uma landing page para uma hamburgueria artesanal, desenvolvida como projeto para portfólio. A página apresenta o cardápio digital com opções de pedidos, design responsivo e navegação intuitiva entre as seções: Início, Sobre, Cardápio e Pedido.

🎯 Funcionalidades
✅ Página inicial com destaque dos hambúrgueres

✅ Página "Sobre" com história e valores da hamburgueria

✅ Cardápio completo com 8 opções de hambúrgueres

✅ Página de pedido com formulário e resumo

✅ Botões "Fazer pedido" com navegação para página de pedido

✅ Design responsivo para todos os dispositivos

✅ Navegação intuitiva entre páginas

✅ Rodapé com links e informações de contato

🚀 Tecnologias Utilizadas
HTML5 - Estrutura semântica

CSS3 - Estilização e responsividade (arquivos separados)

JavaScript - Interatividade (simulação de pedido)


Git - Controle de versão
