Sobre o Projeto
O HotBurguer é uma landing page para uma hamburgueria artesanal, desenvolvida como projeto para portfólio. A página apresenta o cardápio digital com opções de pedidos, design responsivo e navegação intuitiva.

🎯 Funcionalidades
✅ Cardápio interativo com 6 opções de hambúrgueres

✅ Botões "Fazer pedido" com simulação de carrinho

✅ Design responsivo para todos os dispositivos

✅ Navegação entre páginas (index, sobre, img, cardápio, pedido)

✅ Hero banner com chamada para ação

✅ Rodapé com links e informações de contato

🚀 Tecnologias Utilizadas
HTML5 - Estrutura semântica

CSS3 - Estilização e responsividade

JavaScript - Interatividade (simulação de pedido)

Git - Controle de versão
